# Advanced RAG Practice Project

A hands-on project for learning and implementing Advanced Retrieval Augmented Generation (RAG) techniques using LangChain, ChromaDB, FAISS, Weaviate, Cohere Rerank, and Sentence Transformers.


## Setup

### Create Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Register Jupyter Kernel

```bash
pip install ipykernel

python -m ipykernel install --user --name=advanced-rag
```

---

## Run Notebook

```bash
jupyter notebook
```

or

```bash
code .
```

Select Kernel:

```text
advanced-rag
```
